## 🔧 Common Issues & Fixes

**❌ GUI doesn't appear after executing**
> Ensure you are fully loaded into a Prison Life server before executing. Press
> `Right Shift` or your executor's GUI toggle key. Restart both Roblox and your
> executor if the interface doesn't appear.

**❌ Admin commands not working**
> Admin functionality operates client-side for most commands. Some server-level
> actions may be restricted depending on the server's configuration. Ensure you're
> running the latest build with the updated admin module.

**❌ Auto arrest not triggering**
> Confirm that you are playing as a **guard** — the auto arrest module only
> activates for the guard team. Also check that the detection radius in the
> settings panel is set above zero.

**❌ Kill all doesn't affect all players**
> Some players in the server may have lag compensation or brief invulnerability
> frames. Re-trigger the module after a short delay for full coverage.

**❌ Aimlock not tracking targets**
> Open the GUI and verify the FOV circle size and target priority settings.
> Switching from **Nearest** to **Crosshair** targeting mode can improve accuracy
> in cluttered environments.

**❌ Script not working on Android**
> Confirm you're using a Level 7 mobile executor and running Android 10 or higher.
> Re-paste the script after fully loading into the game. Use the dedicated Android
> build from the install section.

**❌ Anti-virus flags the script**
> Obfuscated Lua scripts are routinely flagged by heuristic AV engines — this is a
> false positive. The full source code is available in this open-source GitHub
> repository for your review before executing.
